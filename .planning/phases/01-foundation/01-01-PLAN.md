  <action>
    1. Initialize shadcn/ui:
       ```bash
       npx shadcn@latest init --yes
       ```
       If prompted for base color, choose `slate` or `zinc`. The primary violet is applied via CSS variable override, not the baseColor.

    2. After shadcn init completes, edit `src/app/globals.css` to override the primary colour variables. Find the `:root` block and update:
       ```css
       --primary: oklch(0.499 0.252 278.7);     /* #7c3aed violet */
       --primary-foreground: oklch(0.985 0 0);   /* near-white */
       ```
       Also in the `.dark` block:
       ```css
       --primary: oklch(0.638 0.252 278.7);     /* lighter violet for dark mode */
       --primary-foreground: oklch(0.985 0 0);
       ```
       Add a comment at the top of the override block: `/* PotPlanner violet theme — primary #7c3aed */`

    3. Create `vitest.config.mts`:
       ```typescript
       import { defineConfig } from 'vitest/config'
       import react from '@vitejs/plugin-react'
       import tsconfigPaths from 'vite-tsconfig-paths'

       export default defineConfig({
         plugins: [tsconfigPaths(), react()],
         test: {
           environment: 'jsdom',
           globals: true,
           setupFiles: './src/tests/setup.ts',
         },
       })
       ```

    4. Create `src/tests/setup.ts`:
       ```typescript
       import '@testing-library/jest-dom/vitest'
       ```
       Also install the jest-dom matchers: `npm install -D @testing-library/jest-dom`

    5. Verify `src/app/layout.tsx` has correct font imports and metadata. Update the metadata title to "PotPlanner" and description to "Household budgeting app".

    6. Run `npx tsc --noEmit` to confirm everything still compiles.
  </action>

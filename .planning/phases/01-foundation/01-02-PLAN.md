    2. **users** table:
       - `id`: serial, primary key
       - `email`: varchar(255), not null, unique
       - `passwordHash`: varchar('password_hash', 255), not null
       - `createdAt`: timestamp('created_at'), defaultNow, not null

    3. **accounts** table:
       - `id`: serial, primary key
       - `name`: varchar(255), not null
       - `initialBalancePence`: integer('initial_balance_pence'), not null, default(0)
       - `createdAt`: timestamp('created_at'), defaultNow, not null

    4. **pots** table:
       - `id`: serial, primary key
       - `name`: varchar(255), not null
       - `allocatedPence`: integer('allocated_pence'), not null, default(0)
       - `rollover`: boolean('rollover'), not null, default(false) — comment: `// v2-ready`
       - `createdAt`: timestamp('created_at'), defaultNow, not null

    5. **bills** table:
       - `id`: serial, primary key
       - `name`: varchar(255), not null
       - `amountPence`: integer('amount_pence'), not null
       - `frequency`: frequencyEnum('frequency'), not null
       - `potId`: integer('pot_id'), references pots.id — nullable (null = potless bill)
       - `nextDueDate`: timestamp('next_due_date'), not null
       - `createdAt`: timestamp('created_at'), defaultNow, not null

    6. **billSplits** table:
       - `id`: serial, primary key
       - `billId`: integer('bill_id'), not null, references bills.id
       - `memberName`: varchar('member_name', 255), not null
       - `percentage`: integer('percentage'), not null — comment: `// 0-100, stored as integer`

    7. **transferHistory** table — add block comment: `// Append-only: rows are NEVER updated or deleted`
       - `id`: serial, primary key
       - `sourceType`: varchar('source_type', 50), not null — comment: `// 'account' | 'pot' | 'external'`
       - `sourceId`: integer('source_id') — nullable
       - `destinationType`: varchar('destination_type', 50), not null
       - `destinationId`: integer('destination_id') — nullable
       - `amountPence`: integer('amount_pence'), not null
       - `description`: text('description') — nullable
       - `createdAt`: timestamp('created_at'), defaultNow, not null

    8. **debts** table:
       - `id`: serial, primary key
       - `name`: varchar(255), not null
       - `balancePence`: integer('balance_pence'), not null
       - `interestRate`: integer('interest_rate'), not null — comment: `// basis points (2500 = 25.00%)`
       - `minimumPaymentPence`: integer('minimum_payment_pence'), not null
       - `createdAt`: timestamp('created_at'), defaultNow, not null

    9. **savingsGoals** table:
       - `id`: serial, primary key
       - `name`: varchar(255), not null
       - `targetPence`: integer('target_pence'), not null
       - `potId`: integer('pot_id'), references pots.id — nullable
       - `createdAt`: timestamp('created_at'), defaultNow, not null

    CRITICAL RULES:
    - Every monetary column uses `integer()` — NEVER `numeric()`, `real()`, or `decimal()`
    - No table has a `currentBalance` or `balance` column that is meant to be updated in place
    - The `transferHistory` table is append-only (document in comments)
    - `import 'server-only'` is the FIRST import in the file

  <action>
    1. Create `src/lib/db/index.ts`:
       ```typescript
       import 'server-only'
       import { drizzle } from 'drizzle-orm/node-postgres'
       import * as schema from './schema'

       export const db = drizzle(process.env.DATABASE_URL!, { schema })
       ```

    2. Create `drizzle.config.ts` at project root:
       ```typescript
       import 'dotenv/config'
       import { defineConfig } from 'drizzle-kit'

       export default defineConfig({
         out: './drizzle',
         schema: './src/lib/db/schema.ts',
         dialect: 'postgresql',
         dbCredentials: {
           url: process.env.DATABASE_URL!,
         },
       })
       ```

    3. Create `scripts/migrate.ts`:
       ```typescript
       import 'dotenv/config'
       import { drizzle } from 'drizzle-orm/node-postgres'
       import { migrate } from 'drizzle-orm/node-postgres/migrator'

       const db = drizzle(process.env.DATABASE_URL!)

       async function main() {
         console.log('Running migrations...')
         await migrate(db, { migrationsFolder: './drizzle' })
         console.log('Migrations complete')
         process.exit(0)
       }

       main().catch((err) => {
         console.error('Migration failed', err)
         process.exit(1)
       })
       ```

    4. Create `.env.example` at project root with all required environment variables:
       ```
       # Database

       POSTGRES_USER=potplanner
       POSTGRES_PASSWORD=potplanner

       # Redis
       REDIS_URL=redis://localhost:6379

       # App
       NODE_ENV=development

       ```

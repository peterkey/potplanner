1. Install ESLint and eslint-config-next as devDependencies:
   ```bash
   npm install -D eslint@latest eslint-config-next@latest
   ```

2. Create `eslint.config.mjs` at repo root with this exact content:
   ```javascript
   import { defineConfig, globalIgnores } from 'eslint/config'
   import nextVitals from 'eslint-config-next/core-web-vitals'

   const eslintConfig = defineConfig([
     ...nextVitals,
     globalIgnores([
       '.next/**',
       'out/**',
       'build/**',
       'next-env.d.ts',
     ]),
   ])

   export default eslintConfig
   ```

   Key decisions per CONTEXT.md:
   - Use `eslint-config-next/core-web-vitals` only -- NO TypeScript-aware rules (no `eslint-config-next/typescript`)
   - Use ESLint 9 flat config format (`eslint.config.mjs`) -- NOT `.eslintrc.json`
   - Import `defineConfig` and `globalIgnores` from `'eslint/config'`
   - Ignore `.next/`, `out/`, `build/`, and `next-env.d.ts`

3. Add two scripts to `package.json` scripts section:
   - `"lint": "eslint ."`
   - `"test": "vitest run"`
